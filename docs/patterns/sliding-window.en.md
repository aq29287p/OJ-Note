# Sliding Window

## 概念說明

**滑動窗口（sliding window）** 亦即在陣列中建立一個可移動的範圍，並在範圍內塞入滿足條件的元素，每次循環便將右指針向前移動，並根據情況移動左指針。

## 特徵規律

1. 陣列/字串
2. 最長或最短連續
